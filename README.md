# Incluya Demo

Demo web estática de Incluya para mostrar la experiencia general del producto en navegador.

## Qué incluye

- portada marcada claramente como demo;
- módulos navegables para `Inicio`, `Oír`, `Conversar`, `SOS` y `Ajustes`;
- capturas reales de la app;
- lightbox para ver imágenes completas.

## Deploy

Este proyecto está pensado para desplegarse directamente en Vercel como sitio estático.

## Nota

No representa la app final ni un entorno productivo. Es una experiencia de demostración para explorar el concepto.
